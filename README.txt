Le dossier image contient le projet image réalisé durant le semestre 4
Avant de le compiler, il faut ajouter les libréries présentes dans le dossier lib dans le classpath
Puis compiler le code java et enfin l'executer
Une fenetre ca s'ouvrir contenant un menu qui vous permet d'importer une image , de la modifier, de l'envoyer
et de l'enregistrer
